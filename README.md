# demo-cotizaciones

Prototipo de cotizaciones en **un solo archivo HTML** para correr local.

## Uso
1. Descarga `demo-cotizaciones.html`
2. Ábrelo con doble clic en tu navegador
3. Crea cotizaciones, guarda en `localStorage`, imprime o exporta a PDF

## Incluye
- Número de cotización
- Cliente y empresa
- Ítems con cantidad/precio/descuento
- ITBIS opcional (18%)
- Totales automáticos
- Guardado local y listado de cotizaciones
- Duplicar cotizaciones
- Imprimir / PDF
